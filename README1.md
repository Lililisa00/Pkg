## Теоретическое исследование цветовых моделей

### RGB (Red, Green, Blue)

Аддитивная цветовая модель, используемая в цифровых устройствах отображения. Каждая компонента варьируется в диапазоне от 0 до 255. Цвет формируется сложением трёх потоков света основных цветов.

### XYZ (CIE 1931)

Цветовое пространство, разработанное Международной комиссией по освещению (CIE). Является эталонной моделью, основанной на исследованиях восприятия цвета человеческим глазом. Компоненты X, Y, Z представляют синтезированные координаты, где Y соответствует яркости.

**Формулы преобразования RGB → XYZ:**

r = R/255, g = G/255, b = B/255
r = r > 0.04045 ? ((r + 0.055)/1.055)^2.4 : r/12.92
g = g > 0.04045 ? ((g + 0.055)/1.055)^2.4 : g/12.92
b = b > 0.04045 ? ((b + 0.055)/1.055)^2.4 : b/12.92

X = r * 0.4124564 + g * 0.3575761 + b * 0.1804375
Y = r * 0.2126729 + g * 0.7151522 + b * 0.0721750
Z = r * 0.0193339 + g * 0.1191920 + b * 0.9503041


### HSL (Hue, Saturation, Lightness)

Модель, основанная на восприятии цвета человеком:
- **Hue (оттенок)** - положение цвета на цветовом круге (0-360°)
- **Saturation (насыщенность)** - интенсивность цвета (0-100%)
- **Lightness (светлота)** - яркость цвета (0-100%)

## Техническая реализация

### Архитектура приложения

Веб-приложение реализовано с использованием:
- **HTML** - структура интерфейса
- **CSS** - стилизация и визуальное представление
- **JavaScript** - логика преобразования цветов и обработка событий

Структура интерфейса

Приложение содержит следующие основные блоки:

Блок RGB - поля ввода и ползунки для компонентов R, G, B (0-255)
Блок XYZ - поля ввода для компонентов X, Y, Z
Блок HSL - поля ввода для Hue (0-360), Saturation (0-100%), Lightness (0-100%)
Визуализатор цвета - прямоугольная область отображения текущего цвета
Информационная панель - отображение текущих значений во всех моделях

Механизм синхронизации

Реализована система предотвращения циклических обновлений с помощью флага isUpdating. При изменении значений в любой модели:

Активируется флаг блокировки обновлений
Выполняются преобразования в другие цветовые модели
Обновляются соответствующие поля ввода
Обновляется визуализатор цвета
Сбрасывается флаг блокировки

Достигнутые результаты

Полная функциональность преобразования между моделями RGB, XYZ и HSL
Автоматическая синхронизация всех компонентов интерфейса
Корректная обработка граничных значений и некорректного ввода
Визуальное представление текущего цвета в реальном времени
Пользовательский интерфейс с интуитивным управлением
